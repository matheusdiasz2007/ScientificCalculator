## 📌 Sobre o Projeto

Este projeto consiste no desenvolvimento de uma **Calculadora Científica em Python**, criada para realizar operações matemáticas básicas e avançadas, oferecendo duas modalidades distintas de utilização.

O objetivo do projeto é unir conceitos matemáticos, programação em Python e organização de software, proporcionando uma ferramenta funcional e escalável.

---

## ✨ Funcionalidades

### 🔹 Versão Simples
Calculadora científica tradicional voltada para cálculos rápidos.

Recursos:

- ➕ Soma
- ➖ Subtração
- ✖️ Multiplicação
- ➗ Divisão
- √ Raiz quadrada
- xʸ Potenciação
- Logaritmos
- Funções trigonométricas
- Constantes matemáticas
- Operações científicas gerais

Execução:

```bash
python calculadora_simples.py
```

---

### 🔹 Versão Completa (Com Gráficos)

Versão avançada que adiciona recursos visuais para análise matemática.

Recursos adicionais:

- 📈 Plotagem de gráficos matemáticos
- 📊 Visualização de funções
- 🔬 Operações científicas avançadas
- 🎯 Interface expandida para exploração matemática

Execução:

```bash
python calculadora_completa.py
```

## 🚀 Instalação

Clone o repositório:

```bash
git clone https://github.com/matheusdiasz2007/Calculadora-Cientifica-com-Python.git
```

Entre na pasta:

```bash
cd Calculadora-Cientifica-com-Python
```

Instale as dependências:

```bash
pip install -r requirements.txt
```
